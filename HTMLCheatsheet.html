HTML Cheat Sheet

This page is my personal reference for common HTML elements. I can check it quickly while I am coding my projects.

Common HTML Elements

Element Example What it does
<h1> ... </h1> <h1>Main Title</h1> Main heading of the page. There should be only one main h1.
<p> ... </p> <p>This is a paragraph.</p> Used for normal blocks of text.
<a> ... </a> <a href="page.html">Go to page</a> Creates a hyperlink to another page or website.
<img> <img src="cat.jpg" alt="Cute cat"> Displays an image. alt text is important for accessibility.
<ul> & <li> <ul>
  <li>Milk</li>
  <li>Bread</li>
</ul> 		Creates an unordered list with bullet points.
<ol> & <li> <ol>
  <li>First</li>
  <li>Second</li>
</ol> 		Creates an ordered list with numbers.
<div> ... </div> <div class="box">Content</div> Generic container used to group other elements (block level).
<span> ... </span> <span class="highlight">word</span> Inline container used to style part of a text.
<header> <header>Top area</header> Semantic tag for the top of a page (logo, navigation, etc.).
<main> <main>Main content here</main> Semantic tag for the main content of the page.
<footer> <footer>Copyright text</footer> Semantic tag for the bottom section of the page.
<section> <section>Group of content</section> Used to group related content into sections.
<nav> <nav>Navigation links</nav> Used to wrap the main navigation links of the site.
<table> <table>rows and columns</table> Container for tabular data (rows and columns).
<form> <form action="send.php" method="post">...</form> Used to send user input data to a server.

Notes

Remember: Every HTML document should start with <!DOCTYPE html> and have the structure <html>, <head>, and <body>.
